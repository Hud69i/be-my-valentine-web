# be-my-valentine-web
# 🌹 Surpresa Romântica: Pedido de Namoro Interativo

Este é um projeto simples, leve e visualmente encantador desenvolvido em HTML, CSS e JavaScript puro (Vanilla JS). Ele foi criado para ser uma forma criativa e moderna de fazer um pedido de namoro ou celebrar uma data especial.

## ✨ Funcionalidades
- **Nebulosa Estelar:** Um fundo animado que simula o espaço sideral.
- **Janelas Estilo Retrô:** Interface que remete a sistemas operacionais clássicos (estilo Windows 95/98).
- **Botão "Não" Fujão:** O botão de recusa foge do cursor, tornando o "Sim" a única opção lógica (e divertida).
- **Jardim de Tulipas Digital:** Após o "Sim", um jardim de flores coloridas cresce processualmente na tela.
- **Sequência de Mensagens:** Um sistema de janelas que guia a pessoa por uma narrativa emocionante.

## 🚀 Como usar
Se você quer usar este projeto para a sua namorada ou pessoa especial, siga estes passos:
1. Faça um **Fork** deste repositório.
2. Altere o arquivo `index.html` para personalizar o nome (mude de "Gabriele" para o nome dela).
3. Personalize as frases nas janelas `msg1`, `msg2`, etc.
4. Hospede gratuitamente no **GitHub Pages**.

## 🛠️ Tecnologias
- HTML5
- CSS3 (Animações, Keyframes e Flexbox)
- JavaScript (Manipulação de DOM e lógica de aleatoriedade)
